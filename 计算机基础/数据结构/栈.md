# 一、定义

- 一种可以实现“先进后出”的存储结构。
- 栈类似于堆箱子。只可以对顶部操作。

# 二、分类

- 静态栈：底层是数组
- 动态栈：底层是链表

# 三、算法

- 出栈
- 压栈

# 四、应用

- 函数调用。如，f()调用g(),g()调用h(),h()调用i()。
- 表达式求值。两个栈来执行，一个栈存储数字，一个栈存储运算符。
- 中断。
- 内存分配。
- 缓冲处理。
- 迷宫

# 五、补充
- 内存
    + 静态内存：栈分配。如，局部变量
    + 动态内存：堆分配。如，malloc()
- 栈和堆
    + 分配内存方式不一样
        * 栈：入栈出栈
        * 堆：堆排序
