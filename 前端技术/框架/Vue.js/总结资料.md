webpack打包工具构建大型项目，vue-cli脚手架工具，快速构建项目

//安装node.js/npm

//全局安装vue-cli
npm install -g @vue/cli

//创建项目
vue create <todolist>

//运行
cd <todolist>
npm run serve

    main.js 入口文件
    App.vue 单模块/文件组件，包含组件所有内容

/components 各类(子)组件
